# 利用python爬取某公众号分享的考研数据库，并获取网盘链接和密码

​	   [某公众号](https://mp.weixin.qq.com/s/iID9XLurpY9DOXTc7pjhqA)在文章内公开分享了考研数据库（近几年的录取数据，如录取人数，考试大纲啥的），这本是一件好事。然而他只做一个导航页显示了每个学校资料的链接，如果要获取每个学校的数据都要分别打开再保存至百度网盘保存。我一看，好家伙408所学校，索性写了个python脚本自动获取了所有的网盘链接和密码，再加上github上面大神写的[百度网盘批量转存工具](https://github.com/hxz393/BaiduPanFilesTransfers)（[点我下载](https://meta.appinn.net/t/topic/16995/5)），粘贴url+密码，即可将数据库保存至自己的百度网盘！

## 脚本源代码

```python
import re
import requests

if __name__ == "__main__":
	# 获取数据详情链接的规则
	findLinkRule = re.compile(r'<a target="_blank"(.*?)a>')
	url = "https://mp.weixin.qq.com/s/iID9XLurpY9DOXTc7pjhqA"
	headers = {
		'User-Agent': 'Mozilla/5.0 (Windows NT 6.1; WOW64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/81.0.4044.138 Safari/537.36 NetType/WIFI MicroMessenger/7.0.20.1781(0x6700143B) WindowsWechat(0x6304051b)',
	}
	text = requests.get(url=url, headers=headers).text
	links = re.findall(findLinkRule, text)
	for link in links:
		findDetailedLinkRule = re.compile(r'href="(.*?)"')
		href = re.findall(findDetailedLinkRule, link)[0]

		findDetailedNameRule = re.compile(r'>(.*?)<')
		name = re.findall(findDetailedNameRule, link)[0].replace("&nbsp;", "").replace(" ", "")

		text2 = requests.get(url=href, headers=headers).text.replace("&nbsp;", "").replace(" ", "")
		findDetailedLinkRule = re.compile(r"varmsg_source_url='(.*?)';")
		href = re.findall(findDetailedLinkRule, text2)[0]
		findCodeRule1 = re.compile(r'提取码(.*?)</strong>')
		code1 = re.findall(findCodeRule1, text2)[0].replace("</span>", "").replace(":", "").replace("：", "")

		findCodeRule2 = re.compile(r'>([0-9A-Za-z]{4})<')
		code2 = re.findall(findCodeRule2, text2)

		# print(name, end=" ")
		print(href, end=" ")
		if len(code1) > 0:
			print(code1, end=" ")
		if len(code2) > 0:
			print(code2[0], end=" ")
		print()
```
